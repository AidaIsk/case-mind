# CaseMind

CaseMind — это система для анализа KYC/AML кейсов с использованием AI.

## Что делает система
- анализирует кейсы
- формирует Decision Note (CDD / EDD / Reject)
- объясняет решение
- выявляет логические ошибки

## Стек
Python • OpenAI API • Prompt Engineering

## Статус
В разработке (focus: reasoning quality & decision consistency)
